# Asset Management & Portfolio Optimization

This project analyzes mutual fund performance and constructs optimized investment portfolios using Python.  
It applies key financial concepts such as risk–return trade-offs, efficient frontier modeling, and Sharpe ratio analysis.

---

## Project Overview
- Evaluated mutual fund returns, volatility, and diversification metrics.
- Built and optimized portfolios using mean–variance optimization techniques.
- Visualized portfolio performance using the Efficient Frontier framework.
- Applied asset pricing models to compare fund efficiency and risk-adjusted returns.

---

## Tools & Libraries
- **Python** (Pandas, NumPy, Matplotlib, SciPy, statsmodels)
- **Jupyter Notebook**
- **Portfolio Optimization** and **Data Visualization**
